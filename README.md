# Alerts V2

Intelligent alerts and insights prototype for CommerceIQ Sales Agent (AllyAI).

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Script | Command |
| ------ | ------- |
| Dev | `npm run dev` (Next.js with webpack) |
| Build | `npm run build` |
| Start | `npm start` |
| Lint | `npm run lint` |
